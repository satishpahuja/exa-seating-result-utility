This application may process examination-related files issued to schools in connection with examinations conducted by the Central Board of Secondary Education (CBSE).

For official and authentic government information, users should refer directly to:

  • CBSE Official Website: https://www.cbse.gov.in

3. Information We Collect

Exa does not collect, store, transmit, or share any personal or sensitive user data.

The app operates entirely offline.

Files selected by users (such as result .txt files or LOC PDF files) are processed locally on the device and are never uploaded to any server.

4. How the App Works

The application allows users to:

  • Process examination result .txt files
  • Generate seating plans from LOC PDF files
  • Create Excel and PDF reports for internal school use

All processing occurs exclusively on the user's device.

Generated reports are saved in the device’s Downloads folder or a location chosen by the user.

5. Permissions

The app may request access to device storage only to:

  • Select input files
  • Save generated output files

These permissions are used strictly for core functionality and for no other purpose.

6. Third-Party Services

Exa does not use:

  • Analytics services
  • Advertising SDKs
  • Tracking tools
  • Cloud storage services
  • External servers

No user data is shared with any third party.

7. Children's Privacy

This app is intended for use by school administrators, educators, and authorized educational institutions.

It does not target children under the age of 13 and does not knowingly collect any data from children.

8. Data Security

Since Exa does not collect or transmit personal data, all information remains stored locally on the user’s device.

Users are responsible for securing their own devices and files.
